# 自由古巴 Cuba Libre

## 配方

材料            | 备注
-------------- |----------------
白朗姆酒        | 1.5 oz
可乐            | 4.5 oz

## 准备

* 冰镇玻璃杯
* 冰块

## 步骤

* 依次放入冰块, 白朗姆酒, 可乐
* 后搅拌即可（辅料有气体的不能摇！）
