# 螺絲刀 Screwdriver

## 配方: 伏特加+橙汁（柑橘類果汁可以替换） （推薦1:3或1:2）

## 準備

* 冰鎮玻璃杯
* 冰塊

## 步驟

* 依次放入冰塊, 伏特加, 橙汁
* 后攪拌即可

## 特點

* 簡單
* 酸甜度由果汁決定
* 易入口，女士推薦

## PS

* SKYY的vodka挺常見 新手選擇恐懼症的話推薦
