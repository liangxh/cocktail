# 玉子酒

## 配方

* 材料: 鸡蛋一个 蜂蜜一大勺 牛奶200ml 日本酒(清酒)35ml(此处用白兰地代替)

## 步骤

* 打鸡蛋
* 牛奶 蜂蜜和酒加到锅里加热至沸腾
* 加入鸡蛋搅拌即可

## 特点

* 甜而不腻
* 在热牛奶中酒依然能保持相对的低温，能喝出一种很微妙的层次感

# 备注

* 日本民间治疗感冒饮品
