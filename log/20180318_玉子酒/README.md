# 玉子酒 Tamagozake

## 配方

材料            | 备注
-------------- |----------------
日本酒          | 1 oz
牛奶            | 6 oz
鸡蛋            | 1 个
蜂蜜            | 1 大勺

## 步骤

* 打鸡蛋
* 牛奶 蜂蜜和酒加到锅里加热至沸腾
* 加入鸡蛋搅拌即可

## 特点

* 甜而不腻
* 在热牛奶中酒依然能保持相对的低温，能喝出一种很微妙的层次感

# 备注

* 日本民间治疗感冒饮品
