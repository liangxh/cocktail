# 橙色微风 Mandarin Orange Breeze

## 配方


材料          | 备注
------------ |----------------
日本酒        | 1.5 oz
苏打水        | 2.5 oz
橙汁          | 2 oz
冰块          | 2块

## 步骤

* 冰镇酒杯
* 放入冰块
* 依次倒入日本酒, 橙汁, 苏打水
* 用调酒勺快速搅拌

## 備注

* 清酒有一點苦, 配合橙汁可以緩解苦味
* 清酒酒味不明顯, 但有點后勁, 屬於喝著喝發現喝多那種
* 苏打水主要是讓雞尾酒最終帶有氣泡口感
